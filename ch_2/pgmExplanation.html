Code Example

In the given example, the user is prompted for their name, which is stored in the name variable. The code then outputs a greeting to the user.
For example, if the user entered Michael Mayhew, then the program would respond with It's nice to meet you, Michael Mayhew! 


    var name
    name = prompt("Hello! What's your name?")
    alert("It's nice to meet you, " + name + "!")
